
VSCode 필수 단축키 (Windows 기준)

📁 파일/탭 관련
- 새 파일 열기: Ctrl + N
- 새 탭에서 파일 열기: Ctrl + 클릭
- 파일 탐색기 열기: Ctrl + Shift + E
- 최근 파일 빠르게 열기: Ctrl + P
- 사이드바 접기/펼치기: Ctrl + B

✍️ 코드 편집
- 복사한 줄 바로 아래 붙여넣기: Shift + Alt + ↓ / ↑
- 한 줄 삭제: Ctrl + Shift + K
- 줄 이동: Alt + ↑ / ↓
- 여러 줄 커서 선택: Alt + 클릭 or Ctrl + Alt + ↓ / ↑
- 코드 자동 정렬: Shift + Alt + F
- 자동완성 보기: Ctrl + Space

🔍 탐색/검색
- 현재 파일 내 검색: Ctrl + F
- 전체 프로젝트에서 검색: Ctrl + Shift + F
- 특정 단어 전체 바꾸기: Ctrl + H
- 정의로 이동: F12
- 정의 위에 미리 보기: Ctrl + K → Ctrl + I
- 변수/함수 이름 바꾸기: F2

🧰 디버깅 & 터미널
- 통합 터미널 열기/닫기: Ctrl + ~
- 디버그 시작: F5
- 디버그 중단: Shift + F5
- 한 줄씩 실행 (Step over): F10

🌈 기타 꿀기능
- Command Palette (모든 명령 검색): Ctrl + Shift + P
- 다크/라이트 테마 변경: Ctrl + K → Ctrl + T
- 설정 열기: Ctrl + ,
- 미니맵 보기/끄기: Ctrl + Shift + P → Toggle Minimap

🍎 macOS 기준 단축키 비교
- 자동 정렬: Shift + Option + F
- 터미널 열기: Ctrl + ~
- 커맨드 팔레트: Cmd + Shift + P
- 파일 찾기: Cmd + P
- 검색: Cmd + F
